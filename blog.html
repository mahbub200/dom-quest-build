What is the Document Object Model (DOM)?

DOM stands for Document Object Model.represents the structure of a document as a tree of objects. It is a platform interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.Its a powerful thing we can do many more things as well.

How do you select an element from the DOM?

There are mainly 5 ways

  • 1. document.getElementById()
  • 2. document.getElementsByClassName()
  • 3. document.getElementsByTagName()
  • 4. document.querySelector()
  • 5. document.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element instead of multiple listeners on individual child elements. This allows events to be handled at a higher level in the DOM tree.
